##Chatroom
A simple chatroom app that uses HTML5, CSS3 JavaScript, and jQuery library for the front-end; and ExpressJS, NodeJS, socket.io in the backend.

![Imgur](http://i.imgur.com/sB9QFyM.jpg)

##How this works
An alert will show up when you open http://localhost:8080 and will prompt you to enter your name.<br>
![Imgur](http://i.imgur.com/WNAXKhN.jpg)

Once you enter your name and your friend enters their name, the two of your names will show up.<br>
![Imgur](http://i.imgur.com/RCDR7Rh.jpg)

You can start chatting with your friend!

##Getting started
**Set Up**<br>
Clone this repo to your desktop and run `npm install` to install all of the dependencies.

**Usage**<br>
Once all the dependencies are installed, you can run `npm start` to start the application. This will fire up the backend. You should be able to launch this website via http://localhost:8080.

Once you do the above, you should be all set!
